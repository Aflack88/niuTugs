# NIU Tugs Website

Official website for the NIU Tugs tradition - Northern Illinois University's legendary tug-of-war competition.

## About

NIU Tugs is a long-standing tradition within NIU Greek Life where fraternities and sororities compete in intense tug-of-war matches. The event builds brotherhood/sisterhood, raises funds for philanthropy, and creates lasting memories.

## Features

- **Information Section**: Overview of the tradition, rules, and what makes NIU Tugs special
- **Schedule**: Upcoming tournament dates and events
- **Results**: Past champions and competition outcomes
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Setup

1. Clone this repository
2. Open `index.html` in a web browser
3. No build process needed - pure HTML, CSS, and JavaScript

## Deployment

### GitHub Pages
1. Push code to GitHub repository
2. Go to Settings > Pages
3. Select main branch as source
4. Site will be live at `https://yourusername.github.io/niu-tugs`

### Other Hosting
Upload all files to any web hosting service. The site is static and requires no server-side processing.

## Customization

### Update Schedule
Edit the schedule section in `index.html` (around line 75)

### Update Results
Edit the results section in `index.html` (around line 115)

### Change Colors
NIU colors are defined in `styles.css`:
- `--niu-red: #BA0C2F`
- `--niu-black: #000000`
